Media placeholders

- Add poster images and thumbnails in these folders:
  - brand/: hero reels and identity visuals
  - headshots/: creator photos
  - portfolio/: thumbnail images for each portfolio item (referenced via the `poster` field in src/data/portfolio/*.json)
- Keep image sizes optimized for mobile:
  - Thumbnails target: 720x1280 or 1080x1920, WebP preferred
  - Hero poster target: 1600px wide max
- Media uploaded through the CMS editor lands in src/assets/ in the repository.
